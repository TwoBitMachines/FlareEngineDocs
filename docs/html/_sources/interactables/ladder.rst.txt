Ladder
++++++

The humble ladder is used for climbing. Add this component to an empty game object.

.. image:: ../images/interactables/LadderGif.gif
   :align: center
   :width: 100%
   
|

.. note::
 The player's Ladder ability must be enabled to interact with ladders.

.. list-table::
   :widths: 25 100
   :header-rows: 1

   * - Property
     - 

   * - Size   
     - The width and height of the ladder.

   * - Position On Awake
     - If enabled, the ladder will reposition itself on Awake in case it was moved.
