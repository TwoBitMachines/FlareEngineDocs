Passenger
+++++++++

In order for the main player to ride a vehicle, it must contain this ability. As of right now, the only way to mount a vehicle 
is to jump on top of it.

.. list-table::
   :widths: 25 100
   :header-rows: 1

   * - Property
     - 

   * - Layer
     - The layer the vehicle belongs to. You can create a Vehicle Layer and use that instead of using Player.

   * - Exit
     - The button that must be pressed to exit a vehicle.
